# DALY-Explain

This repository hosts a comprehensive document on Disability-Adjusted Life Years (DALY) and its role within the Global Burden of Disease (GBD) framework. The document provides a structured understanding of how population health is quantified by integrating both mortality and morbidity into a single metric.

It further explores the mathematical and conceptual foundations of key modeling tools such as DisMod II, CODEm (Cause of Death Ensemble Model), and DisMod-MR, explaining how they are used to generate internally consistent epidemiological estimates and robust cause-specific mortality predictions.

Designed as a teaching-oriented resource, this work bridges theory and application by covering:

Core concepts of DALY (YLL and YLD)
Mechanistic disease modeling using differential equations
Statistical modeling and ensemble approaches for mortality estimation
The evolution from early GBD methods to modern frameworks

This repository also provides a web-based interface for easy reading and access to the full document.
